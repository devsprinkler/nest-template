# Nest.js 템플릿

## 환경

> Node.js v16.15.0
>
> Linux 또는 MacOS

## 설치

```bash
$ npm install
```

## 서버 빌드

```bash
$ npm run build
```

## 서버 시작

```bash
# 빌드 없이 시작
$ npm run start

# development 환경
$ npm run start:dev

# production 환경
$ npm run start:prod
```
